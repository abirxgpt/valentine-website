# GIF Image Guide

## Where to Place GIFs

All GIF files should be placed in: `assets/images/`

## Required GIFs

### 1. Initial Happy GIF
**Filename**: `gif1-happy.gif`
- **Mood**: Happy, hopeful, cute
- **Suggestions**: Cute teddy bear smiling, happy puppy, cute cat with hearts

### 2-7. Progressive Sadness GIFs
**gif2-confused.gif** - Slightly confused/puzzled
**gif3-sad.gif** - Getting sad
**gif4-crying.gif** - Crying, very sad
**gif5-pleading.gif** - Pleading, puppy eyes
**gif6-heartbroken.gif** - Heartbroken
**gif7-hopeful.gif** - Still hopeful, little smile

### 8. Celebration GIF
**Filename**: `celebration.gif`
- **Mood**: Extremely happy, celebrating
- **Suggestions**: Dancing, confetti, fireworks, party

## Where to Find GIFs

### Giphy (giphy.com)
1. Visit https://giphy.com
2. Search for terms like:
   - "cute teddy bear happy"
   - "sad puppy"
   - "crying cat"
   - "pleading eyes"
   - "celebration party"
3. Click the GIF
4. Click "Download" or right-click → Save Image As
5. Rename to match the filenames above

### Tenor (tenor.com)
1. Visit https://tenor.com
2. Search for similar terms
3. Download and rename

## Quick Download (Recommended):

Search these exact terms on Giphy:
1. `cute bear happy love` → gif1-happy.gif
2. `confused cute` → gif2-confused.gif
3. `sad puppy` → gif3-sad.gif
4. `crying cat` → gif4-crying.gif
5. `pleading puppy eyes` → gif5-pleading.gif
6. `heartbroken sad` → gif6-heartbroken.gif
7. `hopeful smile` → gif7-hopeful.gif
8. `celebration party confetti` → celebration.gif

## Optimization

After downloading, compress GIFs to reduce file size:
- Use https://ezgif.com/optimize
- Aim for <500KB per GIF
- Quality: Medium to High

## Alternative: Using Your Own Images

Don't have GIFs? You can use static images (.jpg, .png):
- Just name them with `.gif` extension
- Or update `config.js` to point to `.jpg` or `.png` files

## Note

The website will work WITHOUT GIFs - you just won't see the images. All other features (buttons, confetti, animations) will work perfectly!
